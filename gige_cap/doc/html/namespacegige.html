类型定义说明

typedef std::function<void(void*)> gige::LoopBody

主循环的循环体

LoopBody是一个递归函数构成的状态机。
因为C++不方便直接使用递归，本意是写作 run = run();
因语法不支持故改为 run(&run);
每次执行时，除了按当前的逻辑执行一帧，还会修改run为下一帧要做的事。

在文件 gige_common.h 第 60 行定义.

typedef std::function<LoopBody(GlobalData)> gige::LoopContinuation

主循环的连续

“连续”就是指接下来会发生的事。
因为一些状态的构造函数希望知道后续要执行的动作(成功之后做什么，失败后做什么)， 所以需要传入LoopContinuation。
相应状态的LoopBody会调用LoopContinuation产生一个新的LoopBody并将其绑定到run。
